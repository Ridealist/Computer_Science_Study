# 2.3.1. 네트워크 기기의 처리 범위

- 네트워크 기기는 계층별로 처리 범위 나뉨
- 상위 계층을 처리할 수 있는 기기는 하위계층까지 처리 가능 __But, 반대는 불가__

> ex1. L7 스위치는 애플리케이션 계층 처리 = 그 밑에 모든 계층 프로토콜 처리 가능

> ex2. AP는 물리 계층밖에 처리하지 못함

- 애플리케이션 계층: `L7 스위치`
- 인터넷 계층: `라우터`, `L3 스위치`
- 데이터링크 계층: `L2 스위치`, `브리지`
- 물리 계층: `NIC`, `리피터`, `AP`
