# 👀 옵저버 패턴
## 옵저버 패턴이란?
주체가 어떤 객체의 상태 변화를 관찰하다가 상태 변화가 있을 때마다 메서드 등을 통해
옵저버 목록에 있는 옵저버들에게 변화를 알려주는 디자인 패턴

## 쉬운 설명
현장체험학습일이 되었습니다. 조장은 비상 연락망에 조원들 이름을 적으세요.
선생님 지시사항이 생기거나 바뀌면 조원들에게 알리도록 하세요.
